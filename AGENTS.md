# Creative Destruction Website Instructions

## Brand
Creative Destruction is a premium AI transformation brand.
The site must feel cinematic, dark, intelligent, and brand-specific.

## Core visual direction
- Near-black background with subtle cyan/blue atmospheric gradients
- Premium typography
- Strong visual hierarchy
- Minimal but high-impact composition
- No generic SaaS feel
- No crypto aesthetics
- No boxed dashboard-style visuals unless explicitly requested

## Hero rules
- Do not use a standard left-text / right-image hero
- Hero must be a single immersive composition
- Headline and visual should feel integrated
- The transformation concept must be visible:
  destruction -> transition -> intelligence
- No framed image cards in the hero
- Prefer centered or layered compositions
- Use subtle motion only

## Typography
- Headings: Sora
- Body: Inter
- Strong hierarchy, premium spacing, controlled line-height

## UI rules
- Avoid generic feature-card look
- Avoid heavy borders unless subtle
- Use glow and gradients sparingly
- Every section must feel intentional

## Build behavior
- Reuse existing components when helpful
- But replace layout architecture when the existing one is fundamentally generic
- Do not preserve a weak structure just because it already exists

## Quality bar
The result should feel closer to OpenAI / Apple / high-end AI studio
than to a SaaS landing page template.
